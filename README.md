# DIP 大作业仓库（2026）

这个仓库用于存放我的 DIP 课程大作业代码、文档与结果展示。

## 给读者看的内容

- 项目简介：说明课题目标与方法
- 目录结构：快速了解代码与文档位置
- 运行方式：环境、安装、执行步骤
- 实验结果：图片、指标与结论
- 更新记录：每次迭代的主要变化

## 推荐目录结构

```text
.
├── README.md              # 项目总览（优先维护）
├── docs/                  # 详细文档、报告
├── src/                   # 核心代码
├── data/                  # 数据（可放下载说明，不直接提交大文件）
└── results/               # 实验结果与可视化
```

## 使用 VS Code 实时更新仓库

1. 在 VS Code 中打开本仓库文件夹。  
2. 编辑 `README.md` 或其他文件并保存。  
3. 使用 Source Control 面板查看变更。  
4. 提交并同步到 GitHub（Commit / Sync Changes）。  
5. 读者刷新仓库页面即可看到最新 README 展示。  

> 小建议：优先先更新 README，再提交代码，读者体验会更好。
